Welcome to Colibri
==================

Colibri is a cutting-edge PyTorch library designed for computational imaging tasks, seamlessly integrating optical systems with the latest deep neural networks. Our goal is to facilitate research in areas requiring optics and neural networks, making state-of-the-art algorithms accessible and easy to use for new researchers.

Features
------------------

- **Optical Systems and Deep Neural Networks**: Implementations of Coded Aperture Snapshot Spectral Imager (CASSI), Single Pixel  (SP), Autoencoder, and Unet for diverse computational imaging tasks.
- **End-to-End Frameworks**: Colibri introduces frameworks for end-to-end processing, simplifying the deployment of computational imaging technologies.

Installation
------------------

To get started with Colibri, install the library using the following steps:

.. code-block:: bash

    pip install -r requirements.txt
